## Data and Privacy Notice

This project is designed for educational and portfolio purposes only.

- Uploaded receipt images are processed locally by the backend using OCR.
- No receipt images, personal data, or user information are committed to source control.
- Database credentials and environment-specific configurations are managed through environment variables and are excluded from the repository.
- The nutrition database included in this project contains only generic grocery items and publicly available nutritional estimates.

Users running this project locally are responsible for managing their own data and database configuration.
